<h2 style="margin-top:0;">Repository Overview</h2>

<p>
This repository contains educational projects developed to explore the
fundamental concepts behind <b>Natural Language Processing (NLP)</b>,
including text preprocessing, feature extraction, vector-space models,
word embeddings, and text classification techniques.
</p>

<p>
The projects combine theoretical analysis with practical implementation
using Python and Scikit-learn, emphasizing conceptual understanding,
mathematical intuition, and comparative experimentation.
</p>

<h2 style="margin-top:0;">Projects Included</h2>

<h3>Text Representation and Basic Text Mining</h3>

<p>
Introductory notebook focused on understanding classical NLP representation
methods such as TF, IDF, TF-IDF, and Bag of Words.
</p>

<p>
The notebook manually computes representation metrics using small sentence
examples and compares them against Scikit-learn implementations to analyze
their mathematical behavior and interpretability.
</p>

<br>

<h3>Text Classification and Embeddings</h3>

<p>
Project dedicated to comparing multiple NLP architectures for text classification,
starting from traditional statistical representations and progressing toward
embedding-based approaches.
</p>

<p>
The project evaluates baseline models, word embeddings, advanced embeddings,
and comparative performance across different representation strategies.
</p>
